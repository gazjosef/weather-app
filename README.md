
# Weather App

## Description

A weather app that provides weather information while changing background depending on time of day and conditions.

## How to use

1. Enter City
2. Enter Country Code

E.g Sydney AU

## Built with

- Open Weather API
- React
- TimezoneDB
- Typescript
